# Jammming 🎵

A React web app that allows users to search the Spotify library, create custom playlists, and save them to their Spotify account.

## Features
- Search songs by title, artist, or album
- View song details: title, artist, album
- Create a custom playlist
- Add/remove tracks to/from the playlist
- Export playlist to Spotify (future feature)

## Technologies
- React.js
- JavaScript
- HTML / CSS

## Future Work
- Connect to Spotify API for live search and saving playlists
